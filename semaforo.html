Modelagem com Conjuntos

Aplicação da Teoria dos Conjuntos para representar os elementos do sistema.

VA = {carroA1, carroA2} = {🚗, 🚕}

VB = {carroB1, carroB2} = {🚙, 🚐}

P = {pedestre1, pedestre2} = {🚶, 🚶‍♀️}

  • carroA1 ∈ VA
  • carroA2 ∈ VA
  • carroB1 ∈ VB
  • carroB2 ∈ VB
  • pedestre1 ∈ P
  • pedestre2 ∈ P

S = VA ∪ VB ∪ P

(VA ∪ VB) ∩ P = ∅

Estados do Sistema

S0 = Via A Verde
S1 = Via B Verde
S2 = Pedestre Ativo
S3 = Emergência

E = {S0, S1, S2, S3}

  • S0: Via A verde — 10000001
  • S1: Via B verde — 00110010
  • S2: Pedestre ativo — 00100111
  • S3: Emergência — 01001000

Regras Lógicas

  • Se P = 1, então o estado será Pedestre ativo.
  • Se A = 1 e B = 0 e P = 0, então o estado será Via A verde.
  • Se A = 0 e B = 1 e P = 0, então o estado será Via B verde.
  • Se A = 1 e B = 1 e P = 0, então a Via A tem prioridade.
  • Se emergência = 1, então o estado será Modo emergência.

Expressões Booleanas:

A ∧ ¬B → S0 (Via A Verde)
B ∧ ¬A → S1 (Via B Verde)
P → S2 (Pedestre Ativo)
E → S3 (Emergência)
