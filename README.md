Só abrir o `semaforo.html` no navegador.

## O que o projeto faz

Simula um cruzamento urbano com semáforos inteligentes. A ideia é mostrar como a teoria de conjuntos e lógica booleana pode ser usada pra modelar um sistema real.

O sistema tem 4 estados:
- **S0** - Via A liberada (padrão)
- **S1** - Via B liberada
- **S2** - Pedestre atravessando
- **S3** - Modo emergência (amarelo piscando)

## Arquivos

- `semaforo.html` - estrutura da página
- `style.css` - estilos e animações
- `script.js` - lógica dos estados
